SetUp win10
******************
download python 3.9.6
+++++++++++++++++++++++
path: `Python3.9.6 <https://www.python.org/downloads/release/python-396/>`_

.. note::

    make sure python and pip are ok.

install Sphinx
+++++++++++++++++++++++++++
.. code::

    pip install Sphinx

install breathe
+++++++++++++++++++++++++++
.. code::

    pip install breathe

install sphinx_rtd_theme
+++++++++++++++++++++++++++
.. code::

    pip install sphinx_rtd_theme

install copy button
+++++++++++++++++++++++++++
.. code::

    pip sphinx_copybutton
